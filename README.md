# My Portfolio

## Stack: 

- React
- JavaScript
- Html
- TailWind CSS
- Styled Components
- Docker
- Docker-Compose

### How to install:

- Git Clone
- With docker and docker-compose installed, the following command moves up the stack:
- make up and open:
- localhost:8001
- Clear the stack:
- make clean
- For the stack:
- make down
